# Magpie Repository for Kodi

![icon](https://github.com/Code-E-Magpie/repository.magpie/blob/main/add-ons/repository.magpie/icon.png)

The official repository for Code-E-Magpie add-ons.

Add the Kodi source:<br/>https://Code-E-Magpie.github.io/repository.magpie/

Use the 'Install from zip file' method to install the Magpie Repository from the Kodi source.

Use the 'Install from repository' method to install add-ons from the Magpie Repository.

Checks for updates and their installation are automatic if 'Install updates automatically' is selected in System Settings and 'Auto-update' is selected in 'Add-on information' for each add-on.

For further details see:<br/>https://kodi.wiki/view/Add-on_manager
